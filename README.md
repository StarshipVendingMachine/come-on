# Starship Vending-machine corp.

> CREATIVITY RULES

스타십벤딩머신은 '인간의 창의성을 지원하는 모든 활동을 지향한다'는 모토 아래
그래픽스 및 비전 기술을 바탕으로 동영상 제작, 공유 문화를 더욱 풍성하고 즐겁게 만들고 있으며,
이를 통해 더 나은 커뮤니케이션이 가능한 미래를 바라봅니다.
다가올 미래를 함께 만들고, 과몰입의 즐거움을 경험했던 분들을 기다립니다.
